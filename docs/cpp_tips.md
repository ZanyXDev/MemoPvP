Атрибут [[nodiscard]] (появился в C++17) — это инструмент проверки кода, заставляющий компилятор выдавать предупреждение, если результат функции, помеченной этим атрибутом, игнорируется. Он используется для функций, где неиспользование возвращаемого значения (например, кода ошибки или ресурса) почти наверняка является ошибкой программиста.
Основные аспекты [[nodiscard]]:
Применение: Можно помечать функции, типы (классы, структуры, перечисления) или даже конструкторы.Действие: Если функция возвращает значение помечается как [[nodiscard]], а вызывающий код игнорирует это значение (не присваивает, не использует в выражении), компилятор выдает предупреждение

Случаи использования:
Функции, возвращающие коды ошибок/успеха (например, errno).Функции, возвращающие ресурс, который нужно освободить (утечка памяти).Константные функции, которые не имеют побочных эффектов и бесполезны без использования их результата.

Пример использования:cpp[[nodiscard]] int computeValue();
void foo() {
    computeValue(); // Предупреждение: результат не используется
    int x = computeValue(); // Ок
}
