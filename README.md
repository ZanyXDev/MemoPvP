A free implementation MemoPvP game (Bluetooth connect_ 

Upstream Author(s):

	ZanyXdev <zanyxdev@gmail.com>

Copyright:  2026 Paul (ZanyXDev)

We just wanted a simple memory game for kids 4+ age. 
No ads, no complicated screens, but a straightforward nice looking, working and open source game.

### Features
- 3 Themes : `Animals` :cat2: `Monsters` :octopus: & `Emojis` :smile:  
- 2 types of difficulties :star2:
- Sounds on winning :musical_note:
- Play with local AI-bot or Remote Player (Bluethooth connection)
